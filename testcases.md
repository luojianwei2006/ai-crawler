# 阶段五：测试验收

> 关联文档：`PRD.md`（v0.1，§8 EARS / §9 边界异常）、`review.md`、`tasks.md`
> 一期边界（已确认）：开发者为主 · 私有化自托管 · 独立子进程 · 单机单租户+手动运行 · 内部 MCP 风格 · 免抽成
> 生成日期：2026-07-21

---

## 1. 测试范围与策略

- **核心闭环（必测）**：安装插件 → 设参 → 选模型 → 运行 → 右侧实时日志 → 内容落库。
- **分层**：功能（正常流）+ 边界 + 异常/故障 + 回归 + 安全/权限 + 合规。
- **环境**：私有化单机（PHP 8.2+/Laravel/MySQL8/Vue3），可选 headless Chrome（T3 确认）。
- **阻断判定标准**（是否阻断上线，见 §4）：核心闭环失败、权限越权、密钥明文泄露、合规缺位 → 阻断。

---

## 2. 测试用例集

### 2.1 正常流程（核心）
| 编号 | 用例 | 步骤摘要 | 期望 |
|------|------|----------|------|
| TC-N01 | 运行闭环 | 安装插件→填参数(网址/Cookie/失效日期/模型/深度)→选模型→点运行 | 子进程启动、右侧实时出日志、任务置"成功"、内容落库 |
| TC-N02 | 模型测试按钮 | 模型管理选模型→点测试 | 回显延迟/返回内容/token 用量/错误码，且复用运行链路 |
| TC-N03 | 插件上架审核 | 开发者上传→系统扫描→管理员审核通过 | 插件进市场"已上架"，可被发现安装 |
| TC-N04 | 开发者上传触发扫描 | 上传 .zip 插件包 | 自动静态扫描生成报告，置"待审核" |
| TC-N05 | 用户改密 | 旧密码校验+新密码强度达标→提交 | 密码 AES-256 更新，旧会话失效 |
| TC-N06 | 插件启停 | 已安装插件→启用/停用 | 状态机正确流转，停用后不可运行 |

### 2.2 边界场景
| 编号 | 用例 | 输入/条件 | 期望 |
|------|------|-----------|------|
| TC-B01 | 爬取深度=0 | depth=0 | 系统按默认 1 处理，不崩 |
| TC-B02 | 爬取深度超大 | depth=9999 | 截断到上限（T8 定值），防失控 |
| TC-B03 | 网址格式非法 | 非 URL / 含空格 | 表单校验失败，就地提示，不提交 |
| TC-B04 | Cookie 失效日期缺失/过期 | expired_at 空或 < 当前 | 运行前阻断，提示刷新 |
| TC-B05 | 模型未选 | model 为空 | 校验失败，提示必选 |
| TC-B06 | 上传非 zip / 超大 | .txt / 超大小限制 | 拒绝上传，提示格式/大小 |
| TC-B07 | 单用户多插件 | 同用户装 N 个并分别启停 | 各自状态独立，互不影响 |
| TC-B08 | 日志海量 | 长任务数千行 | 前端可滚动/暂停，刷新可回溯，不卡死 |

### 2.3 异常 / 故障场景
| 编号 | 用例 | 触发条件 | 期望（对齐 PRD §8 Unwanted） |
|------|------|----------|-----------------------------|
| TC-E01 | 目标站 429 / 封禁 | 高频请求被限流 | 退避重试；超限终止任务，日志标红 |
| TC-E02 | Cookie 过期 / 401 | 登录失效 | 阻断任务，标记插件/参数"失效"，提示刷新 |
| TC-E03 | 插件进程崩溃 / 超时 | 子进程异常退出 | **隔离该进程、回收资源**，任务标"失败"，不影响其他任务 |
| TC-E04 | 安全扫描命中危险函数 | 插件含 exec/shell_exec 等越权 | 直接驳回上架，给风险项清单 |
| TC-E05 | 目标站改版 | DOM 结构变化致解析失败 | 插件返回结构化错误，任务标红**不崩主进程** |
| TC-E06 | 网络中断 | 采集中断网 | 断点重试（一期），失败标红 |
| TC-E07 | 权限越权 | 普通用户访问他人插件/Cookie | 接口 403，UI 层不可见（PRD §8 Ubiquitous） |
| TC-E08 | 模型 key 无效 / 余额不足 | 测试或运行调模型失败 | 回显错误码，任务置失败，不泄露明文 key |
| TC-E09 | 磁盘/内存配额触顶 | 子进程超资源配额 | 按 M5 配额终止，记录并告警 |

### 2.4 回归测试范围
- 用户体系与 RBAC（三角色、改密、状态启用/禁用）
- 模型管理 CRUD + 测试按钮（GLM/DeepSeek/GPT 三厂商切换）
- 插件市场列表/搜索/安装/启停/卸载
- 爬虫引擎静态页抓取 + 链接抽取 + 深度 BFS（JS 渲染若一期启用）
- 实时日志 SSE 推流 + 落库 + 刷新回溯
- 加密存储（密码/Cookie 密文率 100%，日志不打印明文）

## 3. 验收清单（按模块勾选，对应 PRD §8）

- [ ] **M1 用户与权限**：三角色权限隔离；密码 AES-256 密文；越权 403
- [ ] **M2 模型管理**：三厂商可配置切换；测试按钮回显延迟/token/错误码；key 密文
- [ ] **M3 插件协议/上传/扫描/审核**：manifest 钩子+权限声明；上传即扫描；危险函数/越权驳回；审核流转正确
- [ ] **M4 爬虫引擎**：静态页抓取；链接抽取+深度控制；解析失败结构化不崩
- [ ] **M5 子进程沙箱**：单插件故障不拖垮主进程；越权能力拒绝；配额/超时生效
- [ ] **M6 实时日志**：右侧实时刷新、自动重连、刷新可回溯；级别/进度正确
- [ ] **M7 运行闭环**：全链路通；Cookie 过期/429/崩溃按 §8 处理
- [ ] **M8 插件市场/开发**：安装/启停正确；模板可下载；元信息（标题/开发者/版本/更新日期/说明）完整展示
- [ ] **M9 Cookie 管理**：密文存储率 100%；过期阻断；跨租户不可见
- [ ] **M10 合规与埋点**：PRD §7 指标可统计；合规提示显著呈现

## 4. 上线风险清单

| 风险 | 等级 | 是否阻断上线 | 说明 / 缓解 |
|------|------|--------------|------------|
| 核心闭环失败（TC-N01） | 极高 | **阻断** | 必须全部通过 |
| 权限越权（TC-E07） | 极高 | **阻断** | 他人插件/Cookie 可见=严重事故 |
| 密钥明文泄露（TC-E08/M2/M9） | 极高 | **阻断** | 日志/接口不得出明文 key/Cookie |
| 合规缺位（M10） | 高 | **阻断** | 登录态采集无告知/robots=法律风险 |
| 沙箱隔离失效（TC-E03） | 高 | **阻断** | 单插件拖垮主进程不可接受 |
| SSE 不实时/断流（M6） | 中 | 不阻断 | 影响体验，需 PHP-FPM 缓冲/超时调优 |
| 扫描漏报（TC-E04） | 中 | 不阻断(但高优先) | 规则清单(T6)+人工审核兜底 |
| 爬取被封（TC-E01） | 中 | 不阻断 | 深度上限+限速+可选代理 |
| JS 渲染未就绪（T3） | 中 | 视 T3 | 若一期启用则阻断相关用例 |
