## 2. 任务详细（目标 / 依赖 / 验收 / 关联文档）

### M0 脚手架与基础设施
- **目标**：Laravel 11/12 + Vue3(Pinia,Vite) + MySQL8 工程初始化；产出私有化部署镜像（PHP+MySQL，可选预装 headless Chrome）。
- **依赖**：—（前置）
- **验收**：`composer install` + `npm run dev` 可起；MySQL 迁移可跑；Dockerfile/部署文档就绪。
- **关联**：`review.md §4.1`；待确认 T3（是否预装 Chrome）。

### M1 用户与权限体系
- **目标**：`users` 表 + RBAC（管理员/普通用户/开发者，spatie/laravel-permission）；登录、改密（AES-256）、状态启用/禁用。
- **依赖**：M0
- **验收**：三角色权限隔离；密码密文存储（PRD §8 Ubiquitous）；越权接口返回 403。
- **关联**：`PRD.md §4.1/§8`；`review.md §3.2`。

### M2 模型管理（含测试按钮）
- **目标**：`model_providers` 增删改 + 统一 `ChatAdapter`（OpenAI 兼容 GLM/DeepSeek/GPT）；测试按钮发最小请求回显延迟/token/错误码。
- **依赖**：M1
- **验收**：三种模型可配置并切换；测试按钮复用运行链路（PRD §8 Event-driven：模型测试）；密钥 AES-256 存储。
- **关联**：`PRD.md §4.7/§8`；`review.md §4.2/§4.5(T7)`。

### M3 插件协议 + 上传 + 安全扫描 + 审核
- **目标**：定义 `manifest.json`（钩子点/权限声明/Tools/Resources）；上传触发自动安全扫描 → 待审核；管理员审核队列（通过/驳回+原因）。
- **依赖**：M1, **M5**（扫描需在沙箱隔离语境下评估危险函数）
- **验收**：上传即扫描出报告；危险函数/越权声明直接驳回（PRD §8 Unwanted）；审核流转符合状态机（review §3.2）。
- **关联**：`review.md §4.2(manifest)/§4.5(T6)`。

### M4 爬虫引擎
- **目标**：Guzzle+DomCrawler(Goutte) 跑静态页；链接抽取 + 深度 BFS 去重；Panther 跑 JS 渲染页（T3 确认一期）。
- **依赖**：M0
- **验收**：参数含网址/深度；核心统一调度，插件只写解析规则（PRD §4.5/§4.6）；深度上限生效（PRD §9 边界）。
- **关联**：`report.md §四(B03)`；`review.md T3/T8`。

### M5 子进程沙箱
- **目标**：插件在独立子进程执行（proc_open）；权限白名单 + disable_functions + 危险函数禁用；资源配额/超时（T2 定值）。
- **依赖**：M0
- **验收**：单插件故障不拖垮主进程（PRD §8 Unwanted：进程崩溃隔离）；越权能力拒绝（Ubiquitous）。
- **关联**：`report.md §七`；`review.md §4.4/§4.5(T2)`。

### M6 实时日志
- **目标**：任务每产一行日志 → 写 `task_logs` + SSE 推流；前端 EventSource + Pinia 渲染（级别/进度）。
- **依赖**：M0
- **验收**：右侧面板实时刷新、自动重连、刷新可回溯（PRD §4.9/§8 State-driven）。
- **关联**：`report.md §四(B06)`；`review.md T4`。

### M7 运行闭环
- **目标**：运行页（左参数+模型选择+运行 / 右日志）；`crawl_tasks` 状态机；串联 M2/M3/M4/M5/M6/M9。
- **依赖**：M2, M3, M4, M5, M6, M9
- **验收**：选插件→设参→选模型→运行→看日志→落库全链路通；Cookie 过期/429/崩溃均按 PRD §8 处理。
- **关联**：`PRD.md §5主流程/§8`；`review.md §3.3`。

### M8 插件市场与开发
- **目标**：市场浏览/搜索/安装/启停；开发入口下载 PHP 模板（manifest 示例+钩子示例）；展示标题/开发者/版本/更新日期/说明。
- **依赖**：M3
- **验收**：安装/启停状态正确；模板可降低开发门槛；元信息展示完整（PRD §4.4/§4.8）。
- **关联**：`PRD.md §4.4/§4.8`；`review.md §3.1`。

### M9 Cookie 管理
- **目标**：`cookies` 表 AES-256 存储、租户行隔离；运行前校验失效日期；临近/过期阻断并提示刷新。
- **依赖**：M1
- **验收**：密文存储率 100%；过期任务被阻断（PRD §8 Ubiquitous/Unwanted）；跨租户不可见。
- **关联**：`report.md §四(B04)`；`PRD.md §4.6/§8`。

### M10 合规与埋点（横切）
- **目标**：埋点（运行曝光/提交/运行/日志首行/结束/测试/上传/审核）；合规底线（robots/限速/退出/告知前置）。
- **依赖**：各模块
- **验收**：PRD §7 指标可统计；合规提示在用户后台显著呈现（PRD §9 合规底线）。
- **关联**：`PRD.md §7/§9`；`report.md §六`。
